# 蜂群驯化师 - 美术资源需求

## 资源生成方式

使用 AI 生成物品图标，其他资源复用游戏内置。

## 资源需求总览

| 资源类型 | 是否需要自制 | 说明 |
|----------|--------------|------|
| 物品图标 | ✅ AI 生成 | 3 个图标 |
| 地面动画 | ❌ 复用 | 使用现有物品动画 |
| 手持动画 | ❌ 复用 | 使用现有物品动画 |
| 蜜蜂颜色/大小 | ❌ API | 游戏 API 实现 |
| 特效 | ❌ 复用 | 使用内置特效 |
| 音效 | ❌ 复用 | 使用内置音效 |
| UI 面板 | ❌ 复用 | 原生 UI 组件拼接 |

## 需要 AI 生成的资源

### 物品图标（共 3 个）

| 道具 | 规格 | 文件名 |
|------|------|--------|
| 驯蜂笛 | 64×64 px | bee_flute.png |
| 蜂巢袋 | 64×64 px | bee_bag.png |
| 指挥棒 | 64×64 px | bee_staff.png |

### 图标风格描述

#### 1. 驯蜂笛 (bee_flute)
- 类似笛子/排箫的乐器
- 带有蜜蜂/蜂巢元素装饰
- 红/蓝/绿宝石点缀（呼应配方材料）
- 金色基调

#### 2. 蜂巢袋 (bee_bag)
- 类似小袋子/荷包的容器
- 蜂巢六边形纹理
- 金黄色为主色调
- 可加蜜蜂图案或蜂蜜元素

#### 3. 指挥棒 (bee_staff)
- 类似权杖/指挥棒
- 顶端有蜜蜂或蜂巢装饰
- 金色为主（呼应黄金配方）
- 有指挥/统领的威严感

### 图标风格要求

| 要求 | 说明 |
|------|------|
| 风格 | 与饥荒原版美术风格一致（手绘、略带暗黑卡通风） |
| 轮廓 | 清晰，64px 小尺寸下也能辨识 |
| 背景 | 透明（PNG 格式） |
| 色调 | 偏暖色（金黄、橙色），符合蜜蜂主题 |

## 图标转换流程

1. AI 生成 PNG 图标（64×64，透明背景）
2. 使用 Klei 工具转换为 .tex + .xml 格式
   - 工具：ktools 或 Don't Starve Mod Tools（Steam）
3. 放入 `images/inventoryimages/` 目录
